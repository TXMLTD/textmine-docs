Vault will try and identify as many tags as possible based on the document and its detected type. However, you may wish to create your own tags.

## How to manually create a tag

To manually create a tag, switch to the plain text view of a Vault import and hover over the text which corresponds to the tag you'd like to create.

![Screenshot](/images/intercom/vault/creating-manual-fields-in-vault/01-screenshot-2024-04-21-at-11-02-42.png)

Once you've selected the field, it will appear under the Created fields tab on the right hand side.

![Screenshot](/images/intercom/vault/creating-manual-fields-in-vault/02-screenshot-2024-04-21-at-11-03-18.png)
