TextMine allows you to create custom alerts for your documents. These alerts can be for any of the following:

![Screenshot](/images/intercom/document-lifecycle-management/how-to-create-custom-alerts/01-screenshot-2024-04-21-at-15-30-06.png)

Alerts can be sent to multiple people and you can specify a time and day for the alert which will be delivered by email.

## How to create an alert (Video)

To find out how to create an alert, what the following video or follow the step by step guide below.

## Step by step guide on how to create an alert

To create an alert, go to the Alerts tab in the top left of the side bar.

![Screenshot](/images/intercom/document-lifecycle-management/how-to-create-custom-alerts/02-screenshot-2024-04-21-at-15-31-26.png)

You will see existing alerts which you can manage. You can also create new alerts with the create alert button.

The create alert button will ask you to specify the team followed by the document within that team for which you'd like to create an alert.

![Screenshot](/images/intercom/document-lifecycle-management/how-to-create-custom-alerts/03-screenshot-2024-04-21-at-15-32-54.png)

![Screenshot](/images/intercom/document-lifecycle-management/how-to-create-custom-alerts/04-screenshot-2024-04-21-at-15-33-32.png)

After specifying the document you can then specify the type of alert you'd like to receive followed by who should receive it and when.

Alerts can only be sent to users who have access to the document.

![Screenshot](/images/intercom/document-lifecycle-management/how-to-create-custom-alerts/05-screenshot-2024-04-21-at-15-35-03.png)

Click on save once you are happy with the alert. It will now appear in your dashboard.
