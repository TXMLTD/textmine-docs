The admin of an organisation only has access to the Admin portal which can be used to create business units, roles and assign members of the organisation to a role and/or business units.

![Screenshot](/images/intercom/organisation-settings/user-permission-settings/01-screenshot-2025-05-12-at-11-35-17.png)
