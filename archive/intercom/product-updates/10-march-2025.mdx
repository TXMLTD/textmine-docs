We have released a number of key improvements to the platform:

## Multiple reviewers and better reviewer management

It is now possible to have multiple reviewers. An assignee can do the first analysis of Vault tags and a reviewer can double check the work of the assignee.

![Screenshot](/images/intercom/product-updates/10-march-2025/01-screenshot-2025-03-10-at-15-05-02.png)

You can filter the documents in a Vault team by document type, uploader or assignee.

![Screenshot](/images/intercom/product-updates/10-march-2025/02-screenshot-2025-03-10-at-15-04-17.png)

You can also filter the tags by Reviewer in a Vault document:

![Screenshot](/images/intercom/product-updates/10-march-2025/03-screenshot-2025-03-10-at-15-08-38.png)

## New home screen for Vault users

Vault users can now see recently viewed documents and the documents which have been assigned to them.

![Screenshot](/images/intercom/product-updates/10-march-2025/04-screenshot-2025-03-10-at-15-06-51.png)

## Customise Vault columns

You can configure which columns to display in a Vault team by clicking on the Columns icon below Automated insights. The settings will be saved for the next time you visit the team.

![Screenshot](/images/intercom/product-updates/10-march-2025/05-screenshot-2025-03-10-at-15-01-01.png)

## Agile text extraction

This setting allows you to prioritise quick text extraction techniques over more expensive extraction techniques which might take longer. This can be enabled in the Vault settings:

![Screenshot](/images/intercom/product-updates/10-march-2025/06-screenshot-2025-03-10-at-15-15-08.png)
