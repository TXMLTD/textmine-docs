## Folder improvements

We now display folders in the sidebar below the business unit.

![Screenshot](/images/intercom/product-updates/19-november-2025/01-screenshot-2025-11-18-at-09-19-41.png)

In addition to being able to filter by folder, you can now sort the Vault view by Folder.

![Screenshot](/images/intercom/product-updates/19-november-2025/02-screenshot-2025-11-18-at-09-20-59.png)

## Vault editor improvements

The vault editor performance and usability has been improved.

When validating tags, you will be taken to the next tag to be validated automatically.

When a tag presents data in a list format, we will suggest a list editing tool to make the handling of the data easier.

![Screenshot](/images/intercom/product-updates/19-november-2025/03-screenshot-2025-11-18-at-09-24-16.png)

You can search, keep or delete the selection.

## Image Tags

You can now prompt images within documents. Click on create image tag and select the portion of the image you would like the model to analyse.

![Screenshot](/images/intercom/product-updates/19-november-2025/04-screenshot-2025-11-18-at-09-27-59.png)

Once you've selected the image, create and process your question.

![Screenshot](/images/intercom/product-updates/19-november-2025/05-screenshot-2025-11-18-at-09-32-21.png)

You will then see the answers as image tags the editor.

![Screenshot](/images/intercom/product-updates/19-november-2025/06-screenshot-2025-11-18-at-09-35-08.png)

## Chained Tags

A chained tag is a new tag which allows you to answer questions using answers from other tags. To created a chained tag, click on custom tags and enabled chained tag.

![Screenshot](/images/intercom/product-updates/19-november-2025/07-screenshot-2025-11-18-at-09-38-45.png)

This will open a pop up with tags you can use as source tags to your question.

![Screenshot](/images/intercom/product-updates/19-november-2025/08-screenshot-2025-11-18-at-09-39-40.png)

Once you've selected the source tags, create your question.

![Screenshot](/images/intercom/product-updates/19-november-2025/09-screenshot-2025-11-18-at-09-39-45.png)

Once you run the chained tag, the answer will display with a chain icon.

![Screenshot](/images/intercom/product-updates/19-november-2025/10-screenshot-2025-11-18-at-09-37-40.png)

You can run chained tags on any type of tag including previously run chained tags.

## Company search improvements

We now apply fuzzy logic to company name searches to overcome typos, punctuation or abbreviations.

![Screenshot](/images/intercom/product-updates/19-november-2025/11-screenshot-2025-11-18-at-09-49-45.png)

![Screenshot](/images/intercom/product-updates/19-november-2025/12-screenshot-2025-11-18-at-09-49-53.png)

## Timezones

The times in the app are now displayed based on the user's browser. However, times in PDF exports are still based on the server's timezone.

## General Actions

General agent actions can now be applied to folders. This allows the agent to answer questions looking at the extracted data from all documents in a folder.

![Screenshot](/images/intercom/product-updates/19-november-2025/13-screenshot-2025-11-18-at-09-54-27.png)
