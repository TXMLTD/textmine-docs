It is now possible to import files natively from Sharepoint. To do this you will need the following information about your Sharepoint from your Sharepoint/Azure administrator:

-

Client ID

-

Tenant ID

-

Client Secret

-

Site URL

You can save this details in the Vault Settings of the Vault you would like to connect to TextMine:

![Screenshot](/images/intercom/product-updates/30-october-2025/01-screenshot-2025-10-30-at-08-03-12.png)

Test and then save the configuration.

![Screenshot](/images/intercom/product-updates/30-october-2025/02-screenshot-2025-10-30-at-08-35-10.png)

This will then link the Sharepoint with this Vault meaning you'll be able to import files from Sharepoint into this Vault.

To do this, click upload document and select use an integration.

![Screenshot](/images/intercom/product-updates/30-october-2025/03-screenshot-2025-10-30-at-08-35-25.png)

The Sharepoint option will appear if the settings have been configured.

![Screenshot](/images/intercom/product-updates/30-october-2025/04-screenshot-2025-10-30-at-08-35-34.png)

Select which documents you would like to import:

![Screenshot](/images/intercom/product-updates/30-october-2025/05-screenshot-2025-10-30-at-08-35-47.png)

Documents will then be imported into the Vault and go through the usual extraction process.

![Screenshot](/images/intercom/product-updates/30-october-2025/06-screenshot-2025-10-30-at-08-36-04.png)
