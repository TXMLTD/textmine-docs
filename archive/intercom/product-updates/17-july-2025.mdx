## Document Type Improvements

It is now possible to choose which Document Types, including types created by TextMine, should be included in the document classification.

![Screenshot](/images/intercom/product-updates/17-july-2025/01-screenshot-2025-07-17-at-13-39-03.png)

Your active document types are present in the Document Types tab and you can enable/disable them for a specific Vault at any time.

You can click on the document type to view the associated default tags.

![Screenshot](/images/intercom/product-updates/17-july-2025/02-screenshot-2025-07-17-at-13-46-15.png)

The default tags can be enabled or disabled as required.

Document types in the TextMine library can be promoted to your Document Types by selecting the types you wish to promote and clicking on promote.

![Screenshot](/images/intercom/product-updates/17-july-2025/03-screenshot-2025-07-17-at-13-39-44.png)

Once a Document Type is associated with a Vault, it will be used as part of the classification process. As a result, you no longer need to make special configurations in the Vault settings.

![Screenshot](/images/intercom/product-updates/17-july-2025/04-screenshot-2025-07-17-at-13-50-28.png)

## Agent improvements

Agents can now pull 10-Ks, 20-Fs for companies present in a CSV, xls or xslx.

![Screenshot](/images/intercom/product-updates/17-july-2025/05-screenshot-2025-07-17-at-13-52-23.png)

Download the example CSV and complete it with company names.

![Screenshot](/images/intercom/product-updates/17-july-2025/06-screenshot-2025-07-17-at-13-52-52.png)

Review the suggestions and save the agent.

![Screenshot](/images/intercom/product-updates/17-july-2025/07-screenshot-2025-07-17-at-13-52-30.png)

This will then save the companies and submit the latest 10Ks, 20Fs for those companies to the corresponding Vault.

![Screenshot](/images/intercom/product-updates/17-july-2025/08-screenshot-2025-07-17-at-13-54-12.png)
