## Anchor Tags

If you would like the tag to be connected to a different page, you can move it with the anchor icon.

![Screenshot](/images/intercom/product-updates/5-december-2025/01-screenshot-2025-12-05-at-11-15-34.png)

You can then scroll or navigate to the page where you would like to place the tag.

![Screenshot](/images/intercom/product-updates/5-december-2025/02-screenshot-2025-12-05-at-11-15-46.png)

This will then set the tag to the corresponding page and this change will reflect both in the page navigation and pdf export.
