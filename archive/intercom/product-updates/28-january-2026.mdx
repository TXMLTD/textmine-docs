You can now create PDF or Word reports using data extracted from Vault.

Select the documents which should be included in the report in Vault directly and click on the Generate Report.

![Screenshot](/images/intercom/product-updates/28-january-2026/01-screenshot-2026-01-28-at-09-07-49.png)

You'll then be able to create your report request. You can either use an existing template or create a new request from scratch.

![Screenshot](/images/intercom/product-updates/28-january-2026/02-screenshot-2026-01-28-at-08-41-31.png)

![Screenshot](/images/intercom/product-updates/28-january-2026/03-screenshot-2026-01-28-at-08-43-17.png)

You can also save your report request as a template for future reports. Click on Generate Report and the report will start processing.

You can access reports from the reports icon in Vault.

![Screenshot](/images/intercom/product-updates/28-january-2026/04-screenshot-2026-01-28-at-08-59-35.png)

![Screenshot](/images/intercom/product-updates/28-january-2026/05-screenshot-2026-01-28-at-08-59-37.png)

Click on a report to see the results in more detail.

![Screenshot](/images/intercom/product-updates/28-january-2026/06-screenshot-2026-01-28-at-08-48-11.png)

Findings can be copied or you can download versions of the report in Word or PDF.

![Screenshot](/images/intercom/product-updates/28-january-2026/07-screenshot-2026-01-28-at-08-50-20.png)
