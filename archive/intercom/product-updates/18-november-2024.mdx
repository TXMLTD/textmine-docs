TextMine has released a number of new features to improve the user experience. These include:

-

Pagination of documents in Vault

-

The ability to jump to a relevant page on the preview as opposed to the plain text

-

The ability to generate a marked up pdf containing the

## Pagination

Users can go to a specific page or select a field to go to the page which contains the field. Users can also switch between the preview and plain text of a specific page.

![Screenshot](/images/intercom/product-updates/18-november-2024/01-screenshot-2024-11-25-at-12-42-06.png)

## Marked up exports

You can now generate a marked up copy of the preview which can be easily downloaded and shared with relevant stakeholders.

![Screenshot](/images/intercom/product-updates/18-november-2024/02-screenshot-2024-11-25-at-12-42-34.png)

The pdf can be generated with the download button next to the plain text toggle. The fields will be listed on the corresponding page where the model found the answer.
