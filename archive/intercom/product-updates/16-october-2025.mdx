When importing documents into Vault, you can now organise them as folders. Select an existing folder or create a new one.

![Screenshot](/images/intercom/product-updates/16-october-2025/01-screenshot-2025-10-16-at-10-55-40.png)

After the documents have been imported into a folder, you can add the folder column to the view and filter the Vault by that folder.

![Screenshot](/images/intercom/product-updates/16-october-2025/02-screenshot-2025-10-16-at-10-57-53.png)

![Screenshot](/images/intercom/product-updates/16-october-2025/03-screenshot-2025-10-16-at-10-58-15.png)

You can also move documents between folders by selecting them and clicking on the move to folder icon. You will then be prompted to confirm which folder you would like to send the documents to.

![Screenshot](/images/intercom/product-updates/16-october-2025/04-screenshot-2025-10-16-at-10-56-56.png)

You can view all the folders of the Vault under the Folders tab.

![Screenshot](/images/intercom/product-updates/16-october-2025/05-screenshot-2025-10-16-at-11-00-56.png)

You can then click on a folder to see all the documents inside that folder and you can edit the name of the folder if required.

![Screenshot](/images/intercom/product-updates/16-october-2025/06-screenshot-2025-10-16-at-11-01-10.png)

When you submit documents to Legislate, the folders will be carried through.
