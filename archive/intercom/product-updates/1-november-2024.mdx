TextMine now allows users to skip the TextMine default  document processing pipeline in Vault. When uploading a new document to Vault, simply untick the "Keep default TextMine field" check box.

![Screenshot](/images/intercom/product-updates/1-november-2024/01-screenshot-2024-11-01-at-16-09-59.png)

In this setting, TextMine will run the enabled vault team tags against the document.

![Screenshot](/images/intercom/product-updates/1-november-2024/02-screenshot-2024-11-01-at-16-11-18.png)

Once the document has been processed, the type will be listed as unspecified and you click on it to view the extracted information:

![Screenshot](/images/intercom/product-updates/1-november-2024/03-screenshot-2024-11-01-at-16-13-15.png)

![Screenshot](/images/intercom/product-updates/1-november-2024/04-screenshot-2024-11-01-at-16-17-54.png)

Please note that by deselecting the TextMine fields option, the model will not currently classify the document or look to identify signatures.
