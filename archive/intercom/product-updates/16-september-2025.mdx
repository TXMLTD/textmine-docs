Agents can now perform actions based on Vault results. The first action is based on the confidence score of a document and allows you to assign or submit it based on the score. To get started go to an Agent with an existing source and select Add Action.

![Screenshot](/images/intercom/product-updates/16-september-2025/01-screenshot-2025-09-16-at-13-57-30.png)

You can determine whether the agent should run on existing document or only new documents uploaded to the connected Vault. You can configure the conditions and actions to your requirements.

![Screenshot](/images/intercom/product-updates/16-september-2025/02-screenshot-2025-09-16-at-13-57-53.png)

You will see a summary of the results once the agent has finished running.

![Screenshot](/images/intercom/product-updates/16-september-2025/03-screenshot-2025-09-16-at-14-04-53.png)

We are expanding the list of actions including the ability to apply rules against the values extracted by Vault. Get in touch if you would like to discuss your use case and how it can be solved by TextMine's agents!
