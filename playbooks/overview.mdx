Playbooks turn review policies, clause checklists, rule tables, and negotiation guidance into reusable document reviews.

![Playbooks screenshot](/images/intercom/product-updates/26-may-2026/05-screenshot-2026-05-26-at-14-22-25.png)

## What a playbook contains

A playbook contains rules, expected positions, fallback positions, red flags, and review guidance. TextMine can parse playbook content from source text or documents and create structured rules.

## Types

Playbooks can represent classical review playbooks, policies, or master agreement style review guidance.

## Where playbooks run

Playbooks can be run manually, from Workbench, from Vault processing, or as a Workflow step. Runs are tracked as playbook applications with status, reviewed documents, results, and comparisons.
