title: "Create and run a playbook"
description: "Create a playbook and apply it to Vault documents."
---

## Create a playbook

Create a playbook from source text, a checklist, a policy, or a review guide. TextMine parses rules and saves them into a reusable playbook. Scope the playbook to the relevant Vaults when it should only be available for certain document sets.

## Enable or edit

Playbooks can be active or draft. Users with playbook create permissions can edit, enable, disable, or delete playbooks.

## Run a playbook

Run a playbook against selected Vault documents from the Playbooks page, Vault, Workbench, or a Workflow. Use the runs view to inspect status, source, document, output, and comparison details.

## Auto-apply

When configured, playbooks can be applied automatically after Vault document type detection or processing. Workflows can also apply playbooks as part of a larger document process.
