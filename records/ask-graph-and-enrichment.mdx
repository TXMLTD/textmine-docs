description: "Use records as a queryable business knowledge layer."
---

## Ask Records

Ask Records answers questions over structured records, source evidence, and related documents. It is useful for questions such as "which records need review?", "which fields are missing?", or "what evidence supports this value?".

## Graph view

Use the graph view to inspect relationships between records. This is especially useful for connected entities such as companies, agreements, counterparties, obligations, and ownership structures.

## Enrichment

Records can be enriched from configured sources such as web, Wikipedia, SEC, FT Markets, or news sources where enabled. Use enrichment when records need supplemental context beyond the documents already in Vault.

## Import and export

Use import/export when records need to move between TextMine and another data system or review workflow.
