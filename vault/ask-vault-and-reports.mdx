title: "Ask Vault and reports"
description: "Ask questions and generate reports from Vault evidence."
---

Ask Vault answers questions using Vault documents, extracted tags, and optional uploaded reference text.

## Ask a question

Use Ask Vault for quick answers, comparisons, summaries, or extraction questions. If the question names a specific folder, subset, or document group, resolve those documents first rather than asking the whole Vault.

## Create reports

Use reports when you need a saved analysis, table, comparison, or evidence-backed output. Reports can be created from selected documents, search results, or resolved document groups.

## Sources

Good Vault answers should stay close to the source documents and extracted evidence. For important outputs, include the source document names and the fields or text used to reach the answer.
