Document types and tags define what Vault should extract and review. For the broader Vault extraction and editor reference, see [Vault and editor capabilities](/vault/capabilities).

## Document types

Use document types to group documents with similar extraction needs, such as NDAs, service agreements, invoices, purchase orders, utility bills, or custom business documents.

## Default tags and custom questions

Default tags are reusable extraction fields for a document type. Custom questions can be created and run against selected documents or a whole Vault. Workbench can help create tags, run them, wait for processing, and report on the results.

## Image and table extraction

Vault can also support image transcription/classification, embedded image extraction, table extraction, and line item extraction where configured.

## Reprocessing

When document configuration changes, selected documents can be reprocessed so extracted values reflect the latest tags, document type configuration, or processing pipeline.
